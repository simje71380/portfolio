Snake AI

Summer 2023

This project was developed during my free time. It's a replica of the Snake game implemented in Python, designed for playability by both humans and an AI. The AI functionality is powered by the DQN algorithm and encompasses various observation states (further details available on GitHub). PyTorch was used for implementing and training the AI model.
OpenGL Engine

july 2023 - Ongoing

This project entails creating a 3D rendering engine using OpenGL in C++. It's a personal project that I'm actively developing in my free time. The project is an ongoing endeavor with the objective of achieving a rendering engine with the following functionalities:

• Loading and saving 3D scenes

• Importing meshes (created in Blender)

• Managing materials

• Implementing lighting and shading effects

• Enabling camera movement

Additionally, there's a potential plan to implement ray tracing.

Genetic Algorithm

April - June 2022

This project focuses on solving the Assignment, Scheduling, and Routing Problems for employees within a SESSAD (Specialized Educational and Support Service at Home) setting. The project was undertaken as part of the academic curriculum, aiming to address the challenges of optimizing employee schedules and routes efficiently. A comprehensive examination of various operational research algorithms was conducted, leading to the selection of a Genetic Algorithm as the most suitable solution approach. The project involves the implementation of the Genetic Algorithm to effectively solve the given problem.
StudZero

April - June 2022

This project was undertaken during an Android development course, working in a team of two. StudZero is a mobile Android application aimed at assisting students in managing their budget using the 'zero-based' budgeting technique. The application is developed in JAVA using Android Studio, and we utilized Penpot for the design phase. My primary focus was on the design and front-end development of the application, while my teammate concentrated on the database with SQLite and statistics screen of the app.
